## Send-day hygiene

- **Best days:** Tuesday, Wednesday, Thursday.
- **Best times:** 9–11am or 1–3pm in the recipient's local time zone.
- **Avoid:** Monday mornings, Friday afternoons.
- **One-to-one sends only.** No bulk automation.
